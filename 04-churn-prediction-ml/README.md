# 04 - Churn Prediction ML

## Predicción de Fuga de Clientes
Implementación de un modelo de Machine Learning para predecir qué clientes tienen alta probabilidad de abandonar el servicio.

### Métricas de Negocio
- **Precisión:** Para asegurar que los incentivos de retención se den a los clientes correctos.
- **Recall:** Para no perder clientes que realmente se iban a ir.
- **Análisis de Costo-Beneficio:** ¿Cuánto ahorramos al prevenir una fuga vs el costo del incentivo?

### Herramientas
- **Modelo:** RandomForest/XGBoost.
- **App:** Streamlit para simulación de escenarios de churn.
