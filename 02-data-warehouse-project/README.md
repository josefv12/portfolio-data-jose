Design a relational database optimized for business analytics and revenue insights.

# 02 - Data Warehouse Project

## Modelo Estrella y Arquitectura ETL

Explicación detallada del diseño del almacén de datos.

### El Modelo Estrella

- **Tablas de Hechos:** Ventas, Transacciones.
- **Tablas de Dimensiones:** Productos, Tiempo, Clientes, Geografía.

### Proceso ETL

1. **Extracción:** Fuentes de datos diversas.
2. **Transformación:** Limpieza y normalización mediante SQL/Python.
3. **Carga:** Inserción en el Data Warehouse optimizado para analítica.
