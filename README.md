## 🧩 Proyectos

### 🔹 1. Análisis de Ingresos en Retail — Exploración y Segmentación

📁 [Ver proyecto →](./01-retail-revenue-optimization)

Análisis descriptivo del comportamiento de compra de más de **1 millón de transacciones reales** de un e-commerce UK (2009–2011) para identificar oportunidades de crecimiento.

| Pregunta de negocio   | Hallazgo                                          |
| --------------------- | ------------------------------------------------- |
| ¿Los ingresos crecen? | Pico en noviembre 2011 — £1,456,776 en un mes     |
| ¿Dependencia de UK?   | 83% del revenue concentrado en un solo mercado    |
| ¿Cuándo compran?      | Jueves y miércoles son los días con mayor volumen |
| ¿Hay retención?       | 72% de clientes identificados regresan            |

**Stack:** Python · Pandas · Matplotlib · Seaborn · Estadística descriptiva

### 🔹 3. Dashboard Ejecutivo Interactivo de KPIs

📁 [Ver proyecto →](./03-executive-dashboard) · **[🚀 Ver dashboard en vivo →](https://portfolio-data-jose-7kav7yepgv3how7vfbtywu.streamlit.app)**

Dashboard público con análisis de más de **1 millón de transacciones**, filtrable por año y país, con KPIs en tiempo real y 4 módulos de análisis.

| Módulo         | Qué responde                                                                 |
| -------------- | ---------------------------------------------------------------------------- |
| Revenue Trends | ¿Cómo evolucionan los ingresos? ¿Qué mercados y temporadas lideran?          |
| Products       | ¿Qué productos generan más revenue? ¿Cuál es la curva Pareto?                |
| Customers      | ¿Cuántos clientes nuevos por mes? ¿Qué peso tienen los clientes registrados? |
| Returns        | ¿Qué países y productos tienen mayor tasa de devolución?                     |

**Stack:** Python · Streamlit · Plotly · PostgreSQL · Docker
