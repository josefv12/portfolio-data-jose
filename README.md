# Data Science & Analytics Portfolio - Jose Fernandez

Bienvenido a mi portfolio profesional de datos. Aquí encontrarás proyectos que abarcan desde el análisis exploratorio hasta el despliegue de modelos de Machine Learning, siempre enfocados en generar valor de negocio.

## Proyectos Destacados

### 1. [Optimización de Ingresos en Retail](01-retail-revenue-optimization/)

- **Herramientas:** Python, Pandas, Seaborn, Matplotlib.
- **Descripción:** Análisis profundo de comportamiento de compra, segmentación RFM y análisis de cohortes para identificar oportunidades de crecimiento.

### 2. [Proyecto de Data Warehouse](02-data-warehouse-project/)

- **Herramientas:** SQL, Python (ETL).
- **Descripción:** Diseño e implementación de un modelo estrella para centralizar información de ventas y mejorar la eficiencia de las consultas.

### 3. [Dashboard Ejecutivo](03-executive-dashboard/)

- **Herramientas:** Power BI / Streamlit.
- **Descripción:** Visualización interactiva de KPIs críticos para la toma de decisiones estratégicas de nivel C-level.

### 4. [Predicción de Churn (ML)](04-churn-prediction-ml/)

- **Herramientas:** Scikit-learn, XGBoost, Streamlit.
- **Descripción:** Modelo predictivo para reducir la fuga de clientes, analizando el impacto financiero de la precisión vs recall.
