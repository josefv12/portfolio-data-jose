# 03 - Executive Dashboard

## Storytelling: Del Problema a la Solución
Este dashboard permite a los ejecutivos monitorear el pulso del negocio en tiempo real.

### El Problema
Falta de visibilidad unificada sobre los KPIs de ventas y márgenes entre diferentes regiones.

### La Solución
Un tablero interactivo que destaca:
- Comparativa de Revenue vs Target.
- Top 5 productos por margen de contribución.
- Mapa de calor de ventas regionales.

### Resultados
Reducción del tiempo de reporte semanal en un X% y detección temprana de caídas en ventas regionales.
